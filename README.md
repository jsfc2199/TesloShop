#Teslo API
1. Clonar proyecto
2. ```yarn install```
3. Clonar ```env.template``` y renombrarlo a ```.env```
4. Cambiar las varaibles de entorno de acuerdo a lo necesitado
5. Ejecutar el docker compose
```
docker-compose up -d
```

6. Levantar 
```
yarn start:dev
```
